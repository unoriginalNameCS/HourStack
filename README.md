# HourStack

Available at: https://gray-river-014108700.4.azurestaticapps.net
Simple full-stack app with:

- Backend: .NET
- Frontend: React (Vite)

## Run backend

```bash
cd backend
dotnet run
```

## Run frontend

```bash
cd frontend
npm install
npm run dev
```

## Project structure

- backend/
- frontend/
